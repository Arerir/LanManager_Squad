# Remove admin pages from attendee MAUI app

**Date:** 2026-04-08  
**Author:** Circe (MAUI Dev)  
**Status:** Proposed

## Summary

LanManager.Maui is now attendee-only, so admin pages (CheckIn, Attendance, DoorScan) are removed and owned by LanManager.Maui.Crew.

## Rationale

The crew application owns staff workflows; keeping admin pages out of the attendee app prevents duplicated UI and ensures operators use the dedicated crew build.
